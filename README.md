# arXivPaperReader  
· 利用LLM提取、分析arXiv论文，适合需要速览大量论文的时候使用  
## 【注意】
1. 可自行配置翻译api，当前仅测试过百度，测试通过
2. 可自行配置模型api，当前仅测试过deepseek模型，测试通过
3. 访问arXiv不要太频繁，api有频率限制（默认3.1秒间隔）
4. 目前在测试使用大模型批量分析论文的功能时，只进行了小批次测试，问就是主包抠门，大量批次处理结果未知
## 【使用方式】
1. 点击.exe文件运行程序
2. 设置完检索参数后，点击``搜索预览``可获取论文列表
3. 选择不同``分析选项``后点点击``完整分析``可获取相应结果
4. 可选择``不使用ai分析``，这时程序只会拉取论文的基本信息【标题、摘要、作者等】
5. 支持``翻译``和``pdf下载``，下载pdf后文件目录会自动创建output文件夹，可在此路径下面查看，当然程序在可视化页面也提供了打开文件路径入口
